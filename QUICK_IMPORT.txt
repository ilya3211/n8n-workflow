  КАК ИМПОРТИРОВАТЬ В N8N
═══════════════════════════════════════════════════════════════════════════════

1. Откройте n8n
2. Workflows → "+" → Import from URL
3. Вставьте одну из ссылок выше
4. Import → Activate

═══════════════════════════════════════════════════════════════════════════════
  ТРЕБУЕТСЯ BROWSERLESS
═══════════════════════════════════════════════════════════════════════════════

Запустите Browserless через Docker:

docker run -p 3000:3000 browserless/chrome

Или с токеном:

docker run -p 3000:3000 -e "TOKEN=your-token" browserless/chrome
